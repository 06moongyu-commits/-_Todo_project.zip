# 🚀 Vibe Coding Project: Simple ToDo List Web App

## 💡 프로젝트 소개
AI(Cursor IDE)와의 공동 창작 방식인 '바이브 코딩'을 통해 구현된 To-do List 서비스의 MVP(Minimum Viable Product)입니다. 백엔드 서버 없이 브라우저의 Local Storage를 사용하여 데이터를 관리합니다.

## ✅ 구현된 주요 기능
- **할 일 CRUD:** 할 일 추가(Create), 수정(Update), 삭제(Delete) 기능 구현.
- **완료 상태 관리:** 할 일 완료 상태를 체크하고 해제할 수 있습니다.
- **데이터 영속성:** 모든 데이터는 브라우저의 Local Storage에 저장되어 브라우저를 닫았다 열어도 유지됩니다.
- **사용자 편의 기능:** (이전에 구현하셨던 추가 기능이 있다면 여기에 명시 - 예: 기한순 정렬, 알림 기능 등)
- **반응형 디자인:** 모바일 및 데스크톱 환경 모두에 최적화된 UI를 제공합니다.

## 🛠️ 사용한 기술 스택
- **프론트엔드:** HTML, CSS, JavaScript (Pure JS)
- **데이터 관리:** Web Storage API (Local Storage)
- **개발 환경:** Cursor IDE, AI Agent (Claude 또는 GPT-4/5)

## ▶️ 실행 방법
1. 본 GitHub 저장소의 파일을 다운로드합니다.
2. 다운로드한 폴더 내의 **`index.html`** 파일을 웹 브라우저(Chrome, Edge 등)로 엽니다.
3. 별도의 설치 없이 바로 To-do List를 사용할 수 있습니다.
